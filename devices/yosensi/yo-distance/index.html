YO Distance

By Yosensi Sp. z o.o.

YO Distance is a LoRaWAN device for ultrasonic distance measurement. An example of using this device is monitoring the water level in the tank. Additionally, the device contains sensors that measure temperature, relative humidity and an accelerometer that enables to determine whether the device has been installed correctly.
