FTD Network Tester

By Adeunis

The Adeunis FTD (Field Test Device) is a LoRaWAN® network tester that consists of a GPS receiver and a temperature sensor. It checks network coverages and chooses the most appropriate network for ensuring the good positioning of end devices.
