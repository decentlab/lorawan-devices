LoRa Kit 151(Class C ABP)

By HelTec AutoMation

The HelTec LoRa Kit 151 (Class C ABP) is a LoRaWAN® development board based on the TM32L151CBU6MCU and SX1278 LoRa chip (433/470 MHz) that supports standard LoRaWAN protocol.
