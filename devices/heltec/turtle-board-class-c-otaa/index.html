Turtle Board(Class C OTAA)

By HelTec AutoMation

The HelTec Turtle board (Class C OTAA) is a LoRaWAN® development board that supports standard LoRaWAN protocol and RadioShuttle protocol. It allows connecting various sensors through its pins.
