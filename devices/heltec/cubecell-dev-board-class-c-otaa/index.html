HTCC-AB01(Class C OTAA)

By HelTec AutoMation

The Heltec HTCC-AB01 CubeCell – Dev-Board is a LoRaWAN® development board based on ASR605x (ASR6501, ASR6502). The ASR605x chip is integrated with the PSoC® 4000 series MCU (ARM® Cortex® M0+ Core) and SX1262. HTCC-AB01 allows connecting various sensors and supports the Arduino® development environment.
