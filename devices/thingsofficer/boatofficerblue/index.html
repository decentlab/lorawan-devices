BoatOfficer Blue

By ThingsOfficer Gmbh

The BoatOfficer Blue is a tracking device that measures the position of a boat using the integrated GPS tracker. In addition to that, it has an integrated temperature sensor and a vibration sensor. It also supports adding sensor cables for monitoring battery voltage, detecting motor activity, and detecting water. The collected data is transferred via LoRaWAN®.
